# Local Infinity

## Purpose
This project is an experimental web application that aims to provide a user interface where simply scrolling down with the mouse wheel triggers a locally running AI model to generate content.
Speed is the ultimate goal. Speed above quality, speed over size, speed over everything. Must be ultra fast!

## Constraints
- Must run locally on the user's machine
- Must use a locally running AI model
- Must be able to generate content in real-time as the user scrolls
- Works with <= 12GB VRAM

## Content Types
- Text
- Images

## Quality Control
- The length of output text can be short to improve speed.
- The size of the images can be small to improve speed.

## Tech Stack
- Node
- Bun
- Vite
- React
